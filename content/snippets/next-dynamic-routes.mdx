### Incorrect

```tsx
<Link href={`/snippets/slug/${snippet.slug}`}>
  <a>{snippet.name}</a>
</Link>
```

### Correct

```tsx
<Link href={{ pathname: "/snippets/[slug]", query: { slug: snippet.slug } }}>
  <a>{snippet.name}</a>
</Link>
```
